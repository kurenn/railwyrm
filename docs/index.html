Rails Bootstrap CLI

Build your Rails app with confidence from day one.

Railwyrm helps you start with a reliable Rails baseline, then guides you through auth and quality features with an interactive wizard. You focus on your product, and Railwyrm handles the repetitive setup.

Core Command 

bundle exec ruby exe/railwyrm new my_app

Deterministic generation flow with optional interactive wizard.

Motivation

  • Start each project faster without redoing the same setup over and over.
  • Get quality and security tooling in place early, while the app is still simple.
  • Keep app generation reproducible for local development, teams, and CI.

What Railwyrm Is

  • A Ruby CLI for bootstrapping Rails apps with sensible defaults.
  • An interactive wizard for auth and security-related features.
  • A feature lifecycle toolkit for existing apps: install, status, and sync.

How the script works

  1. Create a Rails app with PostgreSQL + Tailwind defaults.
  2. Install baseline gems and project tooling.
  3. Run framework setup (Devise, storage, ActionText, RSpec, CI, Untitled UI).
  4. Apply the wizard selections for auth and sign-in UX.
  5. Track installed features in `.railwyrm/features.yml` for easy lifecycle management.

Usage

Create a new app

bundle exec ruby exe/railwyrm new my_app

Non-interactive mode

bundle exec ruby exe/railwyrm new my_app \
  --interactive=false \
  --path /tmp \
  --devise_magic_link \
  --devise_passkeys

Install features on existing apps

bundle exec ruby exe/railwyrm feature list
bundle exec ruby exe/railwyrm feature status --app /path/to/app
bundle exec ruby exe/railwyrm feature install magic_link --app /path/to/app
bundle exec ruby exe/railwyrm feature install quality --app /path/to/app
bundle exec ruby exe/railwyrm feature sync --app /path/to/app

Features

Default stack

  • PostgreSQL + Tailwind CSS
  • RSpec + GitHub Actions CI
  • Devise + Active Storage + ActionText
  • Untitled UI + Claude on Rails swarm
  • RuboCop, Brakeman, Bullet, Ruby LSP, Dotenv

Wizard features

Pick auth and UI options during `railwyrm new`.

Devise modules

confirmable lockable timeoutable trackable

Auth extensions

  • magic_link via `devise-passwordless` (auto-enables `trackable`)
  • passkeys via `devise-webauthn`

Sign-in layouts

simple_minimal card_combined split_mockup_quote
Tip: you can run non-interactive and still choose features with flags.

Vision

Deterministic by default. AI first-class by choice.

Railwyrm is designed to help teams ship Rails products faster with reproducible setup first, then accelerate feature delivery through AI-ready workflows. The product contract emphasizes deterministic generation, safety, and versioned evolution.

  • Deterministic by default
  • AI is optional but first-class
  • Preview before apply
  • Safety by design
  • Versioned evolution
