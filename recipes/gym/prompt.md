# Gym Recipe Prompt (Reference)

Use this prompt in Codex/Claude against a newly generated Railwyrm app to validate the Gym reference recipe behavior.

## Copy/Paste Prompt

```text
You are validating a Gym Management overlay in an existing Rails app generated by Railwyrm.

Context:
- Rails app includes Tailwind, PostgreSQL, RSpec, Devise, Active Storage, ActionText, untitled_ui, and claude-on-rails.
- Use recipes/gym/recipe.yml as source of truth.

Goals:
1) Validate core models for members, visits, plans, class sessions, and bookings.
2) Validate Pundit authorization baseline for gym workflows.
3) Validate responsive pages for dashboard, members, visits, class sessions, and reports.
4) Validate seed data and public flows.
5) Keep implementation deterministic and maintainable.

Execution:
- Summarize recipe intent
- Verify models/migrations/enums/associations
- Verify controllers/routes/policies
- Verify views and reusable components
- Run `bundle exec rspec`, `bin/rails routes`, and `bin/rails zeitwerk:check`
- Return concise changelog and remaining TODOs
```
