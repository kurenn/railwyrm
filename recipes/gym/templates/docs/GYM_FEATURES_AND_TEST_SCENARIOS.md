# Gym Features and Test Scenarios (Draft)

## Starter Feature Set

- Member management (create, edit, view)
- Visit tracking (check-ins/check-outs)
- Membership plans
- Class sessions and bookings
- Dashboard and reports overview

## Manual Test Scenarios

1. Sign in as staff and open dashboard.
2. Create a new member and confirm it appears in members index.
3. Record a visit for a member and verify it appears on visits page.
4. Create a class session and book a member into that class.
5. Verify schedule page is visible without authentication.
